Detailed Description

template<int ndim, typename T>
class shark::ndim::SparseArray< ndim, T >

A local data structure to store values for specific positions in a global domain. The data structure is sparse: only selection of the positions is available. This is an auxiliary data structure for gather/scatter operations.
